Google Maps API

The link that helped us create a map for our website is: https://developers.google.com/maps/documentation/javascript/ . We followed geocoding and geolocation tutorials from the Google API to find the user's location.

Refuge Restrooms public API

The link for the Refuge restrooms public API is: http://www.refugerestrooms.org/api/docs/#!/restrooms . This API helped us
find the locations of bathrooms in the US.
